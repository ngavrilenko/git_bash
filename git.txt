Создание своего репозитория с таким же именем, как и имя пользователя в GitHub
git init ngavrilenko

Клонирование репозитория на свой компьютер в отдельную папку
git clone git@github.com:ngavrilenko/ngavrilenko.git

Клонирование репозитория https://github.com/testrusau/testrusau на свой компьютер в отдельную папку
git clone git@github.com:testrusau/testrusau.git

Перемещение из documents в директорию testrusau 
cd testrusau 

Копирование содержимого из testrusau в ngavrilenko 
cp -r testrusau ngavrilenko             

Индексирование всех изменённых файлов 
git add . 

Создание коммита с сообщением
git commit -m "Оформление портфолио"  

Отправление изменений в удаленный репозиторий
git push
